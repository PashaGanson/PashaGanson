I build practical AI-agent workflows: voice calls, concierge automation, CRM/reporting pipelines, and small operating systems for real-world work.

Right now I am focused on turning AI agents from chat interfaces into useful operators that can make calls, prepare reports, coordinate tasks, and return clear results.

## Current Focus

- AI agents for personal operations, sales, reporting, and concierge tasks
- Realtime voice-call automation and post-call summaries
- Business reporting pipelines for CRM and operations teams
- Lightweight dashboards and control surfaces for multi-agent workflows
- Learning in public through open-source contributions and small demos

## Open Source

- OpenClaw contribution in review: [private outbound voice-call objectives](https://github.com/openclaw/openclaw/pull/83942)
- Fork: [PashaGanson/openclaw](https://github.com/PashaGanson/openclaw)

The OpenClaw PR adds a separate private objective for realtime outbound calls, so a call can start naturally while the agent still follows the task goal. Example: say a short greeting, then privately pursue a restaurant reservation objective without reading the whole prompt aloud.

## Stack Around Me

TypeScript, Python, Node.js, GitHub automation, LLM agents, realtime voice, CRM/reporting workflows, Telegram/WhatsApp operations, and practical automation around messy business processes.

## Direction

I care about systems that close the gap between intent and execution:

- "Book this"
- "Call them"
- "Summarize what happened"
- "Prepare the report"
- "Track the follow-up"
- "Tell me only what matters"

The goal is not to make demos that look smart. The goal is to make agents that actually finish work.
